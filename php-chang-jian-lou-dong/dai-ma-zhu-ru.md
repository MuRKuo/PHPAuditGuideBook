# 代码注入

### 例子

有些特殊业务使用了eval等代码执行函数

```
<?php eval( 'echo ('.$_GET['a'].');'); //计算器?>
```

正常输入: ?a=9\*9

服务器执行 echo (9\*9);

输出:81

攻击者输入?a=system('whoami')

服务器执行echo (system('whoami'));

成功调用system函数执行命令

实际业务中要尽量避免使用eval这种动态执行代码方法 必要使用时做好过滤
